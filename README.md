# Emotion Detection Web Application

This project is a web-based Emotion Detection Application built using Flask and IBM Watson NLP services.

The application:

- Accepts user text input via a web interface
- Sends the text to the Watson Emotion Prediction API
- Analyzes emotional scores
- Returns detected emotions and the dominant emotion

## Features

- Flask-based web application
-  Emotion analysis using Watson NLP API
- Returns emotion scores: Anger, Disgust, Fear, Joy, Sadness
- Identifies dominant emotion
- Handles invalid input gracefully

## Tech Stack

- Python 3.8+
- Flask
- Requests
- JSON
- IBM Watson NLP Emotion API
